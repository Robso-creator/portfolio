Prazer! Sou o Robson.

Engenheiro de dados e aspirante a cientista de dados. Experiência em desenvolvimento de pipelines de dados desde o ETL até a construção de modelos e deploy em produção.

Habilidades

ETL

Uso de scripts python para extração, tratamento e carregamento em um banco de dados

Modelagem de Dados

Modelar e criar infraestrutura de bancos aplicando técnicas mais usadas no mercado

Ciência de Dados

Construir modelos preditivos e descritivos para trazer inteligencia às tomadas de decisão

Tecnologias

Principais tecnologias com que trabalho:

Apache Airflow é uma plataforma de fluxo de trabalho para agendar, monitorar e gerenciar processos de dados complexos.
Python é uma linguagem de programação de alto nível, interpretada e de propósito geral, conhecida por sua sintaxe simples e legibilidade.
Git é um sistema de controle de versão amplamente utilizado para rastrear alterações no código-fonte durante o desenvolvimento.
Docker é uma plataforma de código aberto que permite a criação, o envio e a execução de aplicativos em contêineres.
Ubuntu é um sistema operacional de código aberto baseado em Linux, desenvolvido e mantido pela Canonical.
AWS é uma plataforma de computação em nuvem oferecida pela Amazon.
